Скетч создан для Arduino LCD KeyPad версии 
https://www.dfrobot.com/wiki/index.php/Arduino_LCD_KeyPad_Shield_(SKU:_DFR0009)
соответственно, ноги D4, D5, D6, D7, D8, D9, D10, AD0 заняты.

База транзистора управления вибромотором подключена через резистор 1k к AD1,
бипер, таким же образом, подсоединен к AD4.

Часы RTC DS1302 подключены следующим образом:
  Rst -  к ноге D2
  Sclk - к ноге D12
  I/O -  через резистор 1k к ноге D3*
  Vcc - к +3.3 V
  
* резистор нужен для работы трёхвольтового RTC с пятивольтовым микроконтроллером

Каналы реле подключены к D11, D13, AD2, AD3, AD5.

adc_key_val[5] - массив для определения нажатой кнопки (зависит от вашей версии
платы, лучше протестить с помощью скетча LCD_keypad_test).

Неточность хода часов можно скорректировать, введя в меню требуемое время кор-
ректировки в секундах за сутки. Но точность RTC DS1302 зависит от температуры.

Обращаю Ваше внимание на то, что в этой версии скетча первый канал отвечает за
подсветку экрана. При вызове меню подсветка горит на полную. Настройка подсвет-
ки производится в подменю "Set backlight" системного меню.

В рабочем режиме, помимо меню (Select), задействованы следующие кнопки:
  Left -  вкл/выкл 1-го канал (освещение) 
  Up -    смена режима подсветки экрана 
  Down -  немедленный запуск кормушки
  Right - вкл/выкл 2-го канала (помпа)
